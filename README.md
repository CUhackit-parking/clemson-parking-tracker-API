# clemson-parking-tracker-API

An API to keep track of parking lot capacities. Created using python with fastapi. 
The API is hosted using Deta at https://api.clemsonparking.tech and https://xe3vmj.deta.dev. The API docs can be accessed at https://api.clemsonparking.tech/docs.  
Run 'python -m uvicorn main:app --reload' to deploy the app locally.
